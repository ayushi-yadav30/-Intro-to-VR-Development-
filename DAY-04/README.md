# Day 4 - Blender Final Scene

## Project Description

This project is a simple 3D workspace scene created using Blender. It includes a table setup with a laptop, lamp, books,frame and a plant.

## Improvements Made

* Added multiple objects to enhance the scene
* Adjusted position, scale, and rotation
* Created a structured environment
* Improved lighting and presentation

## Learning Experience

Through this task, I learned how to:

* Build a complete 3D environment
* Use transformations effectively
* Work with lighting and materials
* Present a scene professionally
